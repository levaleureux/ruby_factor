# ruby_factor
refactor your ruby code with a CLI tool

# Refactoring Pattern
ruby_factor target a list of refactoring pattern

## Expression
1. Extract expression to variable
2. Extract to method


## Variable
1. rename variable

## Methods
1. [x] Remove
2. Rename
3. Move

## Class
1. Slipt (extract methods to new class)
2. Rename
